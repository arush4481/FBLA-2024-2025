Project Description
The Student Finance Tracker is a simple, user-friendly web application designed to help students manage their personal finances. Created for the FBLA 2024-2025 Coding & Programming competition, this tool allows students to track income, expenses, and monitor their spending habits.

Features
Transaction Management: Add, edit, and delete financial transactions
Balance Tracking: View current account balance updated in real-time
Income & Expense Summaries: See total income and expenses at a glance
Time-Based Filtering: View transactions by week, month, or year
Category Management: Organize transactions by categories
Search Functionality: Easily find specific transactions
Responsive Design: Works on desktop and mobile devices
How to Use
Download the Files: Download all three files (index.html, styles.css, script.js)
Open the Application: Double-click on index.html to open in your browser
Add Transactions: Use the form at the top to add income or expenses
View Your Data: See your financial summary and transaction history
Filter & Search: Use the tools to analyze your spending patterns
Technical Details
Built with pure HTML, CSS, and JavaScript
No external libraries or frameworks required
Data stored locally in your browser (no server needed)
Works offline after initial load
Project Files
index.html - The main HTML structure
styles.css - All styling for the application
script.js - All functionality and interactivity
Privacy & Security
All your financial data is stored locally on your device using browser localStorage. No data is sent to any external servers, ensuring your financial information remains private.
